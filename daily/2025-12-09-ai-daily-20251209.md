### **今日AI资讯**

#### **新闻快讯**

1. **谷歌TPU产能暴涨、挑战英伟达主导地位**
   谷歌 **TPU** 产能预测大幅上调，2027/2028 年将分别达到 **500/700 万块**，增幅最高达 **120%**。[摩根士丹利] 认为 **TPU 供应不确定性已基本解决**，谷歌将加速面向第三方数据中心销售芯片，补强 **GCP 业务**。预计每 **50 万块 TPU 可带来 130 亿美元**收入。谷歌此举或撼动 **英伟达GPU垄断格局**。
   ![TPU动态图](https://image.jiqizhixin.com/uploads/editor/a549b595-0426-4d28-8f80-1b97e1819ec2/640.gif)
   [原文链接](https://www.jiqizhixin.com/articles/2025-12-09-9)

2. **通义千问发布 Qwen Code v0.3.0**
   新版引入 **Stream JSON 模式**、三层 **Adapter 架构**、完整 Session 管理，并加入 **中英文双语界面**。安全性强化包括 **20MB 内存安全缓冲**、修复 Windows 编码问题，同时提升跨平台稳定性。生态方面支持 **ModelScope Provider** 并修复多项核心逻辑。
   ![Qwen Code](https://upload.chinaz.com/2025/1209/6390089550748636842208553.png)
   [原文链接](https://www.aibase.com/zh/news/23514)
