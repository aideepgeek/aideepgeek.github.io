今日AI资讯

新闻快讯

  1. Morisot：无需Prompt的AI绘图产品上线。 一款名为 Morisot 的新型AI画图产品正式上线，它旨在解决用户”知道自己看腻了但不知道自己想要什么”的难题。该产品支持用户在不写Prompt（提示词）的情况下进行AI创作，秉持”先看，再选，再改”的理念，强调人类创作应始于视觉体验。 原文链接
  2. OpenAI与Anthropic联手推动MCP Apps提案。 作为 AI Agent 基础设施的 MCP 协议（机器通信协议）社区正式发布了 MCP Apps 提案（SEP-1865），该提案由 OpenAI 和 Anthropic 共同支持。该提案旨在规范对交互式用户界面（UI）的支持，使MCP服务器能够直接向主机提供可视化的操作界面，从而告别传统的纯文本和结构化数据交互限制。 原文链接
  3. 具身智能落地难题：芯片架构需统一认知与控制。 人形机器人要实现大规模应用，必须解决算力架构的限制，目前许多演示系统仍面临平衡性差、反应迟缓或高度依赖人工操控等问题。行业突破的关键在于芯片厂商（如英特尔）需要创新，通过一颗 SoC（System-on-Chip）将智能认知与实时控制统一到同一个架构中。 原文链接
  4. 清华团队提出DAVSP：实现大型视觉语言模型安全对齐。 针对大型视觉语言模型（LVLMs）容易被恶意图像-文本输入诱导给出有害响应的安全风险，清华大学人工智能学院团队提出了 DAVSP（Deep Aligned Visual Safety Prompt）。该方法引入了全新的视觉安全提示与深度对齐策略，能够显著提升模型抵御恶意攻击的效力，同时几乎不损害模型的正常实用性。该工作已被 AAAI 2026 接收为 Oral 论文。 原文链接
  5. Anthropic财务表现亮眼：三年爆赚700亿。 Anthropic（Claude的开发商）在AI竞争中采取了聚焦利润的策略，与其竞争对手OpenAI侧重生态系统（语音、视频、插件）的扩张形成对比。数据显示，Anthropic在三年内取得了高达 700亿人民币 的总利润，并产生了 170亿人民币 的现金流。 原文链接
  6. 顶流设计Agent Lovart接入Nano Banana Pro。 顶流设计师Agent Lovart 现已正式接入 Nano Banana Pro。这一结合极大地增强了设计Agent的风格转换能力，用户只需一句话输入，即可实现风格的剧烈变化，例如将 BlackPink 风格转化为”东北翠花”风格。 原文链接

社交媒体

  1. OpenAI发布企业级AI应用指南与资源。 OpenAI发布了名为「Guides and resources」的集合，为开发者、初创企业和大型企业提供了一系列 11 份指南和资源。这些内容覆盖了从工程实践到领导力决策的AI应用，重点强调“人类+AI”协作，避免过度依赖模型。关键洞察包括：使用编码智能体可加速软件开发生命周期40%-60%（但需人工审查），以及GPT-5在复杂推理上比前代模型优越25%。 原文链接
  2. Redis推出「Redis for AI」加速器及语义缓存课程。 Redis 发布了针对 GenAI 应用的加速器 「Redis for AI」，该工具结合了快速的向量数据库、丰富的生态集成（兼容 LangChainAI 和 llama_index）及全球规模化能力。其核心功能包括高效的 RAG 和语义搜索、用于上下文保留的长期/短期记忆机制、通过捕捉常见查询语义本质来避免重复LLM调用的语义缓存，以及语义路由。同时，Redis团队与 DeepLearningAI 联合发布了新课程：「Semantic Caching for AI Agents」。 原文链接
  3. 深度解析Claude Agent Skills的底层原理。 一篇从第一性原理角度深度剖析 Claude Agent Skills 的文章指出，技能并非传统意义上的同步函数调用，而是一套纯基于提示（Prompt）的元工具系统。技能的作用是”改写上下文”，通过向当前对话动态注入一段带有最高优先级的系统级指令（即“提示扩展器”），从而临时改变Claude的行为模式和可用工具权限。这种异步特性使其特别适合处理需要多步推理和状态管理的工作流。 原文链接
