今日AI资讯

新闻快讯

  1. 脉冲神经网络 (SNN) 团队 Max-Former 实现了性能与能效的双突破，通过校正频率偏置，解决了 SNN 在处理高频细节时的二进制短板，提高了模型精度并实现省电，相关研究由香港科技大学（广州）等团队完成，并将发表于 NeurIPS 2025。 原文链接
  2. Google 宣布 RAG (检索增强生成) 技术面临”死刑”，现在通过 Gemini 的 File Search，工程师只需一行 API 调用，即可将检索、分块、索引和引用等传统复杂流程全部封装，极大地简化了开发流程。 原文链接
  3. Safe Superintelligence Inc. 创始人 Ilya Sutskever 在最新访谈中做出断言：”Scaling 时代已经终结”，AI 社区正在从大模型竞赛转向研究时代，该深度对话在 X 平台上突破了百万浏览量，并得到了诸多业内重量级人物的共鸣。 原文链接
  4. 爆火的 Nano Banana Pro 被发现具备惊人的”时空重现”能力，用户只需提供一个坐标和一个可选时间，它就能拟真生成该时空场景的影像，例如重建 911 事件现场或化身泰坦尼克号幸存者。 原文链接

开源项目

  1. 阿里开源了新项目 ROCK，为智能体打造了一个可以海量复制的”实战演练场”，有效解决了智能体无法在真实环境中进行规模化训练的难题。 原文链接
  2. Black Forest Labs 发布了爆火的开源版视觉模型 FLUX.2（即开源版 Nano Banana），它实现了从演示噱头到专为现实创意工作流程打造的生产力工具的升级，相较于前代 FLUX.1 性能有大幅提升。 原文链接

社交媒体

  1. Indeed 最新报告显示科技行业的招聘岗位遭遇寒冬，特别是数据和分析领域的招聘岗位与疫情前相比减少了 40%，导致该领域的供需严重失衡，竞争因求职者数量增加和生成式 AI 兴起而异常激烈。 原文链接
  2. 有开发者正在跟踪 Claude Code 模型的每一次 Prompt 变化，发现最新版本新增了 EnterPlanMode 和 ExitPlanMode 两个 Tool，允许用户在 .claude/plans 目录下编写 Plan。 原文链接
  3. 一位用户分享经验称，使用 GPT 耗费大半天未能解决的问题，最终被 antigravity (Gemini) 成功解决，表示未来可能主要依赖 Gemini 或 Copilot，并暗示目前 Copilot 尚不成熟。 原文链接
  4. 关于 LLMs 内部机制的观点指出，LLMs 并非进行线性思考，其核心工作是通过嵌入整个提示、执行大规模并行矩阵运算和同步更新概率，基于全局模式选择 token；模型的线性输出仅仅是因为人类语言是线性的。 原文链接
  5. Sam Altman 透露，OpenAI 的首款设备将具有“iPhone 级别”的革命性，但它的设计理念是带来”平静和安宁”，而不是充斥着令人心烦意乱的闪烁灯和通知。 原文链接
